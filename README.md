# Object Detection 

Проект по Computer Vision:
обнаружение и детекция медицинских масок

## Проект — Object Detection

Использованы модели:

- Faster R-CNN
- YOLOv8s

Проведено сравнение моделей по метрикам:

- mAP50
- Precision
- Recall

## Результаты

### YOLOv8s

- mAP50 ≈ 0.89
- Precision ≈ 0.90
- Recall ≈ 0.90

### Faster R-CNN

- mAP50 ≈ 0.88

## Основные технологии

- PyTorch
- Torchvision
- YOLOv8
- Albumentations
